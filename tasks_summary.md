# Завдання з Методів Математичної Фізики (ММФ)

## Завдання 18 — Фундаментальний розв'язок (тепловий імпульс)

| Елемент | Опис |
|---------|------|
| **РЧП** | `u_t = α² · u_xx` на нескінченній прямій |
| **ПУ** | `u(x,0) = δ(x)` — дельта-функція |
| **Формула** | `G(x,t) = 1/√(4πα²t) · exp(-x²/(4α²t))` |
| **Графіки** | 2D-профілі G(x,t) для різних t, залежність max G від t, 3D-поверхня |
| **Лекція Фарлоу** | 5 (формула 5.2), 12 (перетворення Фур'є) |

## Завдання 19 — Задача Коші з довільним імпульсом

| Елемент | Опис |
|---------|------|
| **РЧП** | `u_t = α² · u_xx`, `-∞ < x < ∞` |
| **ПУ** | `u(x,0) = φ(x)` — довільна форма |
| **Формула** | `u(x,t) = ∫ G(x-ξ,t)·φ(ξ) dξ` — згортка з ядром теплопровідності |
| **Імпульси** | Прямокутний, трикутний, Гауссівський |
| **Лекція Фарлоу** | 12 (Фур'є-перетворення задачі Коші) |

## Завдання 20 — Змішана задача з ГУ 1-го роду (Фур'є)

| Елемент | Опис |
|---------|------|
| **РЧП** | `u_t = α² · u_xx`, `0 < x < L` |
| **ГУ** | `u(0,t) = 0`, `u(L,t) = 0` — Діріхле (1-й рід) |
| **Метод** | Розділення змінних → ряд Фур'є за синусами |
| **Формула** | `u(x,t) = Σ A_n · exp(-(nπα/L)²t) · sin(nπx/L)` |
| **Додатково** | Аналіз збіжності ряду, спектр коефіцієнтів |
| **Лекція Фарлоу** | 5 (метод розділення змінних) |

## Завдання 21 — Змішана задача з ГУ різного роду

| Елемент | Опис |
|---------|------|
| **Задача A** | Нейман(0) + Діріхле(L): `u_x(0,t)=0, u(L,t)=0` → cos-функції |
| **Задача B** | Діріхле(0) + Нейман(L): `u(0,t)=0, u_x(L,t)=0` → sin з напівхвилями |
| **Задача C** | Нейман(0) + Нейман(L): обидва ізольовані → cos-функції з A_0/2 |
| **Додатково** | Порівняння швидкості затухання для різних ГУ |
| **Лекції Фарлоу** | 3 (граничні умови), 5-6 (розділення змінних) |

## Завдання 22 — Біжучі хвилі (метод Д'Аламбера)

| Елемент | Опис |
|---------|------|
| **РЧП** | `u_tt = c² · u_xx`, `-∞ < x < ∞` |
| **Формула** | `u(x,t) = ½[f(x-ct) + f(x+ct)] + 1/(2c)·∫ g(ξ)dξ` |
| **Варіанти** | Гауссівський імпульс, прямокутний, удар по струні |
| **Додатково** | Розходження хвиль (ліва/права), 3D-візуалізація |
| **Лекції Фарлоу** | 17-18 (формула Д'Аламбера) |

## Завдання 23 — Коливання закріпленої струни

| Елемент | Опис |
|---------|------|
| **РЧП** | `u_tt = α² · u_xx`, `0 < x < L` |
| **ГУ** | `u(0,t) = 0`, `u(L,t) = 0` — закріплені кінці |
| **Метод** | Розділення змінних → ряд за власними функціями |
| **Формула** | `u(x,t) = Σ sin(nπx/L)·[a_n·sin(ω_n·t) + b_n·cos(ω_n·t)]` |
| **Варіанти** | Відтягнута струна (гітара), комбінація синусів, удар (фортепіано) |
| **Додатково** | Окремі моди коливань з вузлами, 3D-візуалізація |
| **Лекція Фарлоу** | 20 (коливання обмеженої струни, стоячі хвилі) |
